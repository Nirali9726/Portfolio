# OIBSIP_WebDevelopment_Task2 — Personal Portfolio Website

## Objective
To build a personal portfolio website to showcase my skills, projects and experience as a web developer, demonstrating knowledge of responsive design, animations, and modern web development techniques.

## Live Demo
🔗 [View Live](https://nirali9726.github.io/OIBSIP_WebDevelopment_Task2)

## Steps Performed
1. Planned all sections of the portfolio — Hero, About, Skills, Projects, Services, Contact, Footer
2. Created the HTML structure with semantic tags
3. Built a pill-shaped frosted glass sticky navbar with logo, nav links and social icons
4. Designed a hero section with large background name text effect and avatar image overlay
5. Created an About section with photo and bio text in a grid layout
6. Built a skills marquee strip using CSS keyframe animations for seamless looping
7. Added a Projects section with 6 real GitHub projects, screenshots, tech tags and live demo links
8. Created a Services section with 3 cards — Frontend Development, UI Design, Web Applications
9. Built a Contact section with contact details on the left and a working form on the right
10. Added a footer with copyright and social media links
11. Wrote JavaScript for hamburger menu, sticky navbar shadow and scroll reveal animations
12. Made the entire portfolio fully responsive using media queries

## Tools Used
- **HTML5** — Semantic page structure
- **CSS3** — Flexbox, Grid, keyframe animations, transitions, glassmorphism, gradient backgrounds
- **JavaScript** — Hamburger menu toggle, scroll effects, form success message
- **Font Awesome** — Icons throughout the page
- **Google Fonts (Outfit)** — Typography
- **VS Code** — Code editor
- **GitHub Pages** — Deployment

## Outcome
A fully responsive personal portfolio website featuring a frosted glass navbar, hero section with avatar and background text effect, auto-scrolling skills marquee, 6 real project cards with live links, services section, contact form, and footer — built using HTML, CSS and vanilla JavaScript.
